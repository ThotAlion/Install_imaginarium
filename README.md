# Install_imaginarium
Install process for hybrid drone cycle robot.

# Pinout for cycle drone :
- GPIO23 = servo-pan
- GPIO24 = servo-tilt
- GPIO26 = brushless 1
- GPIO19 = brushless 2
- GPIO13 = brushless 3
- GPIO6 = brushless 4

![Raspberry Pi 3 pinout](http://i.imgur.com/JEQzRdo.jpg)
